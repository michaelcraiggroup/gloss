# Welcome to Gloss

Gloss is a distraction-free markdown reader for macOS. Open any `.md` file and read it beautifully rendered — no editing distractions, no clutter.

## Headings & Navigation

Headings like this one get anchor links. Hover to see them, click to copy. The **Inspector** sidebar shows a full table of contents.

## Code Blocks

Syntax highlighting is built in. Hover any code block to reveal a copy button.

```swift
struct ContentView: View {
    var body: some View {
        Text("Hello, Gloss!")
            .font(.title)
    }
}
```

```javascript
// Copy this with one click
function greet(name) {
    return `Hello, ${name}!`;
}
```

## Links & Wiki-Links

Regular [markdown links](https://example.com) open in your browser. Wiki-links like `[[another-file]]` navigate within your vault.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `j` / `k` | Scroll down / up |
| `gg` | Jump to top |
| `G` | Jump to bottom |
| `Cmd+F` | Find in page |
| `Cmd+E` | Export as PDF |
| `Cmd+Shift+E` | Toggle edit mode |

## What's Next?

- Open a **folder** to browse all your markdown files in the sidebar
- Use the **Inspector** for table of contents, frontmatter, and backlinks
- **Star** your favorite documents for quick access
- Try **Edit mode** for a live-preview writing experience
